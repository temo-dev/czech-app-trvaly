# Design System: Sahara (Trvalý Exam) ## Visual Identity: "Sun-Baked Simplicity" A warm, disciplined minimalist aesthetic designed to balance the seriousness of exam preparation with a youthful, motivating energy. --- ## 1. Foundation ### Typography (Google Fonts) - **Primary Heading (Serif):** *EB Garamond* - Used for Page Titles, Section Headers, and Brand Presence. - *Styles:* Medium Italic, SemiBold. - *Scale:* - Display: 40px / 1.1 - H1: 32px / 1.2 - H2: 24px / 1.3 - **Body & UI (Sans):** *Manrope* - Used for functional text, inputs, labels, and micro-copy. - *Styles:* Regular, Medium, SemiBold, ExtraBold. - *Scale:* - Body Large: 18px / 1.5 - Body Medium: 16px / 1.5 - Label/Small: 14px / 1.4 - Micro: 12px / 1.2 (Uppercase for tags) ### Color Palette - **Primary (Core Brand):** `#c2652a` (Sahara Orange) - **Secondary (Action):** `#d88c5a` - **Surface/Background:** `#faf5ee` (Warm Paper) - **Neutral High:** `#1c1917` (Stone-900) - **Neutral Mid:** `#78716c` (Stone-500) - **Neutral Low:** `#d8d0c8` (Warm Stone) - **Success:** `#2d7a4d` (Forest Green) - **Error:** `#b91c1c` (Deep Red) --- ## 2. Components ### Cards - **Base Style:** White background, 24px corner radius, subtle shadow (`shadow-sm`). - **Tonal Style:** `#faf5ee` background, 24px corner radius, border `#d8d0c8/60`. ### Buttons - **Primary CTA:** Solid `#c2652a`, White text, 16px corner radius, font-bold. - **Secondary:** Transparent, border `#c2652a`, text `#c2652a`, 16px corner radius. - **Ghost/Text:** Text `#c2652a`, no border, hover background `#c2652a/5`. ### Progress Visuals - **Circular:** Thick stroke, `#c2652a` for progress, `#d8d0c8` for track. - **Linear:** 8px height, rounded ends, `#c2652a` fill. ### Input Fields - **Default:** Border `#d8d0c8`, background white, 12px corner radius, 16px padding. - **Focus:** Border `#c2652a`, ring shadow. --- ## 3. Feedback & States ### AI Feedback Panels - **Smart Cards:** Bordered cards with color-coded highlights (Green for strong, Orange for needs improvement). - **Icons:** Minimalist, thin-stroke iconography. ### Empty States - **Illustration:** Abstract, line-art or subtle 3D icons. - **Layout:** Centered content, serif heading, body text, primary action. ### Leaderboard Elements - **Top 3 Podium:** Elevated avatars with rank badges. - **Row Item:** Clean list item with avatar, name, points, and streak indicator. --- ## 4. Spacing & Layout - **Gutter:** 24px (Mobile), 48px (Web). - **Vertical Rhythm:** 8px base grid. - **Section Spacing:** 40px - 64px.

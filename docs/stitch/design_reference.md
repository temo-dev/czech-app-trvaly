## 🎨 Sahara Design System

| Token | Value |
|-------|-------|
| Primary | `#c2652a` |
| Background | `#faf5ee` |
| Surface | `#faf5ee` |
| Surface container | `#f2ece4` |
| On background | `#3a302a` |
| On surface variant | `#605850` |
| Outline | `#9a9088` |
| Tertiary | `#8c3c3c` |
| Error | `#c0392b` |
| Headline font | EB Garamond |
| Body/Label font | Manrope |
| Border radius | 8px |
| Color mode | Light (warm) |

**Design philosophy:** Sun-Baked Simplicity — luxury warmth meets disciplined minimalism.
Whitespace is the primary tool. Warm-shifted palette even in grays.
